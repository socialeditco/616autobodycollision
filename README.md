# 616 Autobody & Collision Demo Website

Static multi-page HTML/CSS/JS visual proposal for **616 Autobody & Collision** in Grand Rapids, Michigan.

## Files

- `index.html` — homepage with a two-image fade slider, intro, stats, services, and CTAs
- `services.html` — collision repair, painting, and dent repair service detail sections
- `gallery.html` — GSAP before/after gallery with lightbox
- `about.html` — shop story, placeholder team, values, and certification badges
- `contact.html` — demo quote request and appointment scheduler using flatpickr
- `styles.css` — shared Acura dark/red visual system
- `script.js` — shared drawer, slider, smooth scroll, GSAP, lightbox, flatpickr, and demo alerts

## Notes

This is a visual proposal/demo. Forms do not send email, create CRM records, or add calendar events. They only show demo alerts.

The site can be opened directly from `index.html` or deployed as a static site on Vercel.
