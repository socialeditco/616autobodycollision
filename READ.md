# 616 Autobody & Collision Demo Website

Static multi-page HTML/CSS/JS visual proposal for 616 Autobody & Collision in Grand Rapids, Michigan.

## Files

- `index.html` – homepage with 2-image fade slider
- `services.html` – service detail sections
- `gallery.html` – GSAP before/after gallery and lightbox
- `about.html` – team, history, certifications, values
- `contact.html` – demo quote request and appointment scheduler
- `styles.css` – shared global styles
- `script.js` – shared interactions, slider, drawer, GSAP, lightbox, demo form alerts

Open `index.html` directly in a browser or deploy the folder to Vercel. Forms are demo-only and do not send data.
